# miniguia-napoleon-hill
Miniguia de Estudos sobre Napoleon Hill – Este repositório reúne resumos estruturados, glossário de conceitos e prompts reutilizáveis baseados nas principais obras de Napoleon Hill (A Lei do Triunfo, Pense e Enriqueça e Mais Esperto que o Diabo). O projeto demonstra como a IA pode apoiar a organização do conhecimento e o aprendizado ativo.
